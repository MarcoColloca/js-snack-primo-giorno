Snack Java Script 1° Giorno

Esercizio 1

Dichiara una variabile chiamata number e assegnale il valore 100. Stampa il valore della variabile in console.

Esercizio 2/A

Dichiara una costante chiamata PI e assegnale il valore di 3.14. Stampa il valore della costante in console.

Esercizio 2/B

Prova ad assegnare alla costante chiamata PI il valore di 3.1416. Sai dire cosa succede?

Si ottiene un errore nella console, in quanto una costante può avere solo un valore, inoltre la console non valuterà i comandi successivi.

Esercizio 2/C

Dichiara una variabile chiamata radius e assegnale il valore 8.

Esercizio 3

Dichiara una variabile chiamata name e assegnale il valore “Carlo”. Successivamente, modifica il valore della variabile in “Marco”. Stampa il nuovo valore in console.
